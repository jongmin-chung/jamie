카카오페이 스타일 코드 블록 테스트

다음은 JetBrains Mono 폰트와 Atom One Dark 테마가 적용된 코드 블록입니다:

import sentencepiece as spm

# Initialize SentencePiece model
sp = spm.SentencePieceProcessor()
sp.load('spm.model')

# Encode text to ids
ids = sp.encode_as_ids("Hello world")
print(ids)  # [151, 88]

# Decode ids back to text
text = sp.decode_ids(ids)
print(text)  # Hello world

또한 인라인 코드도 JetBrains Mono 폰트로 스타일링되어 있습니다. 모바일에서는 코드 블록에 가로 스크롤이 적용되고, 넉넉한 패딩이 적용됩니다. 

function calculateTotal(items) {
  return items
    .map(item => item.price * item.quantity)
    .reduce((sum, current) => sum + current, 0);
}

const shoppingCart = [
  { id: 'item1', name: 'Product A', price: 10000, quantity: 2 },
  { id: 'item2', name: 'Product B', price: 25000, quantity: 1 },
  { id: 'item3', name: 'Product C', price: 5000, quantity: 3 }
];

const total = calculateTotal(shoppingCart);
console.log(`Total: ${total}원`);  // "Total: 50000원"
